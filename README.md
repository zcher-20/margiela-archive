# Maison Margiela / Folders

A visual archive of the Maison Margiela folders — spanning the Printed Archive and Digital Archive collections across Artisanal, Anonymity, Tabi, and Bianchetto.

## Features

- Hover-driven gallery with crossfade transitions
- Adaptive nav color based on image brightness
- Lightbox viewer with [NEXT] / [ESC] cursor controls
- Grayscale thumbnail strip with color-on-hover

## Run locally

```
python3 -m http.server 8080
```

Then open [http://localhost:8080](http://localhost:8080).
